# Progresso

Progresso is a productivity app that helps you track your progress on various skills and tasks.

### Linux

Download and run the `.deb` file from the [Releases](https://github.com/6162-code-656C/Progresso-electron) page.

## Usage

- Add and manage your skills and tasks.
- Track your progress and hours spent.
- View your progress on a calendar and chart.
