# Progresso

Progresso is a productivity app that helps you track your progress on various skills and tasks.

## Installation

### Linux

Download and run the `.AppImage` or `.deb` file from the [Releases](https://github.com/yourusername/progresso/releases) page.

## Usage

- Add and manage your skills and tasks.
- Track your progress and hours spent.
- View your progress on a calendar.
